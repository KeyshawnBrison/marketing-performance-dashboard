# Marketing Performance & Customer Behavior Dashboard (Power BI)

This project analyzes marketing campaign performance, customer engagement, and conversion behavior across multiple channels. The goal is to identify the most effective marketing strategies and provide clear, data-driven recommendations using Power BI visualizations and statistical insights.

---

## Project Overview

This Power BI dashboard highlights:

- Which marketing channels deliver the highest **ROI**, **conversion rate**, and **engagement**.
- Monthly trends in customer behavior and campaign performance.
- How customer segments respond to different campaign types.
- Key performance indicators (KPIs) that summarize marketing effectiveness.

The dashboard uses **KPI cards**, **comparison visuals**, **trend analysis**, **drill-down capability**, and **interactive slicers** to help stakeholders explore insights in real time.

---

## Dataset

The dataset includes the following fields:

- `Campaign_Type`
- `ROI`
- `Conversion_Rate`
- `Engagement_Score`
- `Channel_Used`
- `Monthly_Trend`
- Additional marketing metrics used for comparisons

The data is provided in the repository so the PBIX file loads without errors.

## Dashboard Features

### ✔ KPI Cards
- Average ROI
- Average Conversion Rate
- Engagement Score
- Monthly performance indicators

### ✔ Comparison Visuals
- **ROI by Campaign Type**
- **Engagement Score by Channel**
- **Conversion Rates by Channel**
- **ROI vs Engagement Scatterplot**

### ✔ Trend Analysis
- Monthly performance line chart
- YoY or MoM percentage changes

### ✔ Interactivity
- Slicers for Campaign Type, Channel, and Month
- Drill-down enabled charts
- Dynamic filtering across visuals
